Paket adı: Combinia Safe Area and Materials Upgrade

Bu paket iki ana sorunu hedefler:
1) Arka plan kaldırma:
   - kullanıcı fotoğrafı seçtikten sonra önce güvenli alan seçer
   - sonra yalnızca bu alan backend'e gider
   - böylece sandalye / kapı / duvar gibi arka planlar çok daha az karışır

2) Kategoriye göre kumaş/malzeme:
   - Ayakkabı için malzeme seçenekleri değişir
   - Süet, Deri, Nubuk, Rugan gibi seçenekler gelir
   - Çanta ve Aksesuar için de ayrı malzeme listeleri gelir

Not:
Bu paket için şu komutu da çalıştır:
  npx expo install expo-image-manipulator

Sonra:
  npx expo start --clear
